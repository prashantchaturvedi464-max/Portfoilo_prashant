* **Google Fonts & FontAwesome**: For clean typography and intuitive iconography.
✨ Key Features
* **Dynamic Project Gallery**: Showcases projects like *Rozgar Chowk* with live filtering.
* **Responsive Design**: Optimized for all devices—from large desktops to mobile screens.
* **Contact Form**: Functional UI for employers and collaborators to reach out.
* **Modern UI/UX**: Includes smooth scroll animations and a professional color palette.
